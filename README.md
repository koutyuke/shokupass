# Shokupass

茨城工業高等専門学校　PBL実験作成物
学食の食券発見から受け渡しまでを完全無人化するアプリケーション

# Tech stack

* `Web` - Next.js
* `iOS` Mobile APP - Expo
* `API Server Side` - NestJS
* `DB` - Supabase
* `Monorepo` - Turborepo

# Patment
* PayPay

# install
```sh
pnpm i
```

# Develop
```sh
// web
pnpm web dev

// iOS
pnpm mobole ios
pnpm mobile start

// API Server
pnpm api dev
```
